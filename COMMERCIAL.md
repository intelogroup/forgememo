# Commercial & Hosted Options

Forgemem is open source under [Apache-2.0](LICENSE), which permits commercial use, modification, and distribution.

If you want hosted service terms, SLAs, or enterprise support, we offer commercial agreements.

## When You Might Want a Commercial Agreement

- You want contractual SLA and priority support
- You need vendor onboarding, security questionnaires, or procurement support
- You want a hosted or managed deployment for your team

## Tiers

| Tier | What you get | Price |
|---|---|---|
| **Personal** | Local install, BYOK or Ollama, unlimited — no caps, no expiry | Free forever |
| **Solo Managed** | Forgemem handles inference — no API key needed, pay per distill run | $0.02 + token cost per run, or $8/month flat |
| **Sync** | Cloud sync across devices + web memory search UI | $6/month |
| **Solo Pro** | Managed inference + sync bundled | $12/month |
| **Team** | Shared memory pool across devs, team-wide principles | $15/user/month |
| **Enterprise** | SSO/SAML, audit logs, data residency, SLA, priority support | Contact us |

**Personal is genuinely free.** Forgemem runs on your machine. If you use BYOK or Ollama, we pay nothing when you run it — so we charge nothing. No event limits, no seat limits, no expiry.

**Solo Managed** is for developers who want zero-friction inference without managing API keys. You pay a small platform fee per distillation run ($0.02 + actual token cost at Haiku rates), or go flat at $8/month.

**Sync** is for developers who work across multiple machines or want their memory backed up and searchable from anywhere.
